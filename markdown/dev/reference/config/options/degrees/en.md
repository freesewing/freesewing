---
title: degrees
---

For angles, use degrees.

Your degree option should be an object with these properties:

 - `deg` : The default value in degrees
 - `min` : The minimul that's allowed
 - `max` : The maximum that's allowed

```js
options: {
  collarAngle: { 
    deg:  85, 
    min:  60 
    max: 130 
  }
}
```
