---
title: "Tablier Albert : Ce dont vous avez besoin"
---

Pour réaliser Albert, vous aurez besoin de :

- [Fourniture de base pour la couture](/docs/sewing/basic-sewing-supplies)
- Environ 1,25 mètre (1.4 yards) (en fonction de la longueur de votre tablier) d'un tissu approprié ([voir les options de tissu](/docs/designs/albert/fabric))
