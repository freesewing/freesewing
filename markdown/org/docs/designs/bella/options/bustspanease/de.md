Die **Bustumpfschlanke** Option bestimmt, wie viel Leichtigkeit auf den Bustumpfbereich angewendet wird.

<Note>Dies wird das Kleidungsstück nicht leicht machen, sondern nur den Büstenpunkt beeinflussen</Note>
