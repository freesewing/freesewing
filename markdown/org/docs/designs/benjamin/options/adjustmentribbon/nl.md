title: "Aanpaslintje"

![Traditioneel of aanpasbaar](adjustmentribbon.svg)

Met Benjamine kan je een traditionele vlinderdas van vaste grootte maken. And you can make one that incorporates an adjustment ribbon. These are ribbons and pieces of hardware that allow you to change the neck size of the bow tie. Very handy when you have shirts with different neck sizes, or if you have different ease in the collar for some.

<Tip>

Dit is een geweldige optie als je een vlinderdas aan iemand cadeau wil doen.

</Tip>
