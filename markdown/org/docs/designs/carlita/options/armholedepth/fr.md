---
title: "Profondeur d'emmanchure"
---

Cette option contrôle la profondeur de l'emmanchure.

Cette option s'applique à la nouvelle façon v3 de calculer la profondeur de l'emmanchure. If you enable the [legacy armhole depth](/docs/designs/brian/options/legacyarmholedepth) option, this option will not apply, but instead it's legacy version [armhole depth factor](/docs/designs/brian/options/armholedepthfactor) will.
