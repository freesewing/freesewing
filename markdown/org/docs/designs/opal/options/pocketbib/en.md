---
title: "Bib pocket"
---

Toggles whether to include a bib pocket in the pattern.

Note: Setting this option to 'true' will enable the following options:
- Bib pocket on fold
- Bib pocket vertical position
- Bib pocket style
- Bib pocket width
- Bib pocket height
- Bib pocket feature width
- Bib pocket feature height
- Bib pocket curve
