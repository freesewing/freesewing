Distanz beim Schließen des Kragens.

<Note>

Dies ist eigentlich eine Stilwahl, aber eine breitere Kragenspalte (zusammen mit dem Kragenwinkel) kann eine breitere Krawatte (Knoten) aufnehmen.

</Note>
