<Note>

##### Найвища точка плеча

Найвища точка плеча використовується для зняття декількох вертикальних мірок.
Refer to [the hps documentation](/docs/sewing/hps/) for information on how to locate this point.

</Note>

Вимір **від плеча до талії спереду** вимірюється починаючи від найвищої точки плеча вниз через груди до лінії талії.

Для зняття цієї мірки, спочатку [знайдіть найвищу точку свого плеча](/docs/sewing/hps/). Далі вимірюйте від цієї точки вниз до лінії талії.
