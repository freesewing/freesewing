![Die Back-Öffnungs-Option auf Kathrin](./backopening.svg)

Diese Option legt fest, wie weit die Rückenöffnung Ihres Korsetts sein wird. Die Rückenöffnung ist, wo Sie das Korsett zusammensetzen.
