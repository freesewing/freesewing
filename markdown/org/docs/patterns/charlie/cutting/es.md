- Parte **1**: **2 x** _con buenos lados juntos_.

- Parte **9**: **2 x** _con buenos lados juntos_

- Parte **11**: **1 x** si hace una pulsera recta
- Parte **11**: **2 x** _con buenos lados juntos_ si hace una cintura curvada

- Parte **7**: **2 x**
