title: "Back Raise"
- - -

Deze instelling legt de taille op de rug. Onze taille zit niet horizontaal, maar ligt aan de rug. Deze set stelt je in staat om dit in de rug te verhogen als je het nodig hebt voor een goede pasvorm.
