---
title: Unités
---

Cela permet de contrôler les unités utilisées dans votre modèle. FreeSewing supporte les unités métriques et impériales.

Notez qu'en tant qu'utilisateur de FreeSewing, vous pouvez configurer vos unités dans votre compte, et ce sont ces unités qui seront utilisées par défaut.  However, if you want to override those settings on an individual basis -- perhaps you are generating a pattern on behalf of someone else -- this setting allows you to do that.
