---
title: Andere Zeilen
---

Einige Muster haben möglicherweise andere Linien auf ihnen, es gibt 4 zusätzliche Stile:

 - Notiz
 - Markierung
 - Contrast
 - Hilfe

Sie können von Muster-Designern verwendet werden, um zusätzliche Informationen hinzuzufügen, je nach Kontext.
