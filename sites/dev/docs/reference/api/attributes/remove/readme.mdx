---
title: Attributes.remove()
---

The `Attributes.remove()` method removes the attribute values under key and
returns the Attributes object.

## Signature

```js
Attributes attributes.remove(string key)
```

## Example

```js
const attr = new Attributes()
  .add('class', 'classA')
  .add('class', 'classB')
  .remove('class')

// attr holds no data
```
