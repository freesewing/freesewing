## Fit {#fit}
### Back opening {#backopening}

**Controls the opening at the back of the apron**
- Type: **Percentage**
- Default: **10%**
- Minimum: **0%**
- Maximum: **25%**

<Backopening />

### Strap length {#chestdepth}

**Controls the length of the straps**
- Type: **Percentage**
- Default: **22%**
- Minimum: **15%**
- Maximum: **90%**

<Chestdepth />

### Bib length {#biblength}

- Type: **Percentage**
