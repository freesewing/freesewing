## Fit {#fit}
### Adjustment ribbon {#adjustmentribbon}

**Whether or not to include an adjustment ribbon**
- Type: **Boolean**
- Default: **false**

<Adjustmentribbon />

### Collar ease {#collarease}

**The amount of ease at your neck**
- Type: **Percentage**
- Default: **3%**
- Minimum: **0%**
- Maximum: **6%**

<Collarease />

## Style {#style}
### Bow length {#bowlength}

**Length of the bow (when knotted)**
- Type: **Percentage**
- Default: **28%**
- Minimum: **23%**
- Maximum: **33%**

<Bowlength />

### Bow style {#bowstyle}

**Style of the bow**
- Type: **List**
- Default: **butterfly**
- options:
  - diamond
  - butterfly
  - square
  - widesquare

<Bowstyle />

### Collar band height {#collarbandheight}

**The height (width if you will) of the collar band**
- Type: **Percentage**
- Default: **6%**
- Minimum: **5%**
- Maximum: **8%**

<Collarbandheight />

### End style {#endstyle}

**Style of the bow ends**
- Type: **List**
- Default: **straight**
- options:
  - straight
  - pointed
  - rounded

<Endstyle />

### Knot width {#knotwidth}

**Width of the knot**
- Type: **Percentage**
- Default: **7%**
- Minimum: **5%**
- Maximum: **10%**

<Knotwidth />

### Tip width {#tipwidth}

**Width of the tips**
- Type: **Percentage**
- Default: **15%**
- Minimum: **0%**
- Maximum: **20%**

<Tipwidth />
