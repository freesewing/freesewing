## Fit {#fit}
### Length {#lengthratio}

**Controls the length of the bib**
- Type: **Percentage**
- Default: **75%**
- Minimum: **55%**
- Maximum: **85%**

<Lengthratio />

### Neck opening {#neckratio}

**Controls the size of the neck opening relative to the bib size**
- Type: **Percentage**
- Default: **80%**
- Minimum: **70%**
- Maximum: **90%**

<Neckratio />

### Width {#widthratio}

**Controls the width of the bib**
- Type: **Percentage**
- Default: **45%**
- Minimum: **35%**
- Maximum: **55%**

<Widthratio />

## size {#size}
### Head size {#headsize}

**The head circumference you want the bib to accommodate**
- Type: **Percentage**
- Default: **100%**
- Minimum: **10%**
- Maximum: **200%**

<Headsize />
