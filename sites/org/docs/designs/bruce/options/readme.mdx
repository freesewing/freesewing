## Fit {#fit}
### Back rise {#backrise}

**Percentage by which the waist will be raised at the back.**
- Type: **Percentage**
- Default: **5%**
- Minimum: **0%**
- Maximum: **10%**

<Backrise />

### Bulge {#bulge}

**Increase angle to create more room in the front pouch.**
- Type: **Degrees**
- Default: **20°**
- Minimum: **0°**
- Maximum: **40°**

<Bulge />

### Leg stretch {#legstretch}

**For best results, you want to fit your legs a bit more snugly — say no to gaping.**
- Type: **Percentage**
- Default: **40%**
- Minimum: **25%**
- Maximum: **45%**

<Legstretch />

### Stretch {#stretch}

**The amount of negative ease.**
- Type: **Percentage**
- Default: **15%**
- Minimum: **5%**
- Maximum: **25%**

<Stretch />

## Style {#style}
### Leg length bonus {#legbonus}

**Extra length to add to the legs.**
- Type: **Percentage**
- Default: **0%**
- Minimum: **-10%**
- Maximum: **20%**

<Legbonus />

### Rise {#rise}

**Amount to raise the waist. A negative value will lower it.**
- Type: **Percentage**
- Default: **10%**
- Minimum: **0%**
- Maximum: **25%**

<Rise />
