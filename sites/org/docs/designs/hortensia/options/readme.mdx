## Style {#style}
### Handle width {#handlewidth}

**Controls the width of the handle**
- Type: **Percentage**
- Default: **8.6%**
- Minimum: **4%**
- Maximum: **25%**

<Handlewidth />

### Size {#size}

**Controls the overall size of the handbag**
- Type: **Percentage**
- Default: **50%**
- Minimum: **20%**
- Maximum: **200%**

<Size />

### Strap length {#straplength}

**Controls the length of the strap**
- Type: **Percentage**
- Default: **160%**
- Minimum: **75%**
- Maximum: **250%**

<Straplength />

### Zipper size {#zippersize}

**Which size of zipper to use**
- Type: **List**
- Default: **#5**
- options:
  - #3
  - #4
  - #4.5
  - #5
  - #6
  - #8
  - #10
  - invisible

<Zippersize />
