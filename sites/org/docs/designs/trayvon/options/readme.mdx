## Style {#style}
### Knot width {#knotwidth}

**The width of your tie at the knot**
- Type: **Percentage**
- Default: **8%**
- Minimum: **4%**
- Maximum: **12%**

<Knotwidth />

### Length bonus {#lengthbonus}

**How much longer than the default length to make the tie.**
- Type: **Percentage**
- Default: **0%**
- Minimum: **-50%**
- Maximum: **50%**

<Lengthbonus />

### Tip width {#tipwidth}

**The width of your tie at the tip**
- Type: **Percentage**
- Default: **15%**
- Minimum: **5%**
- Maximum: **35%**

<Tipwidth />
