## Conditional Options {#conditional}

### Armhole depth {#armholedepth}

**Controls the depth of the armhole. Higher values make a deeper armhole.**

- Type: **Percentage**
- Default: **2%**
- Minimum: **-10%**
- Maximum: **50%**

<Armholedepth />

## Fit {#fit}

### Chest ease {#chestease}

**The amount of ease at your chest.**

- Type: **Percentage**
- Default: **8%**
- Minimum: **0%**
- Maximum: **20%**

<Chestease />

### Draft for high bust {#draftforhighbust}

**Draft the pattern using the high bust measurement if it is available, instead of using the chest measurement. This can result in a more fitted garment for people with breasts. (Please note that no additional adjustments or shaping is performed. Further manual adjustments may be needed for those with a larger difference between high bust and chest measurements.)**

- Type: **Boolean**
- Default: **false**

<Draftforhighbust />

### Hips ease {#hipsease}

**The amount of ease at your hips.**

- Type: **Percentage**
- Default: **8%**
- Minimum: **0%**
- Maximum: **20%**

<Hipsease />

### Stretch {#stretchfactor}

**Determines the horizontal negative ease.**

- Type: **Percentage**
- Default: **5%**
- Minimum: **0%**
- Maximum: **15%**

<Stretchfactor />

## Style {#style}

### Armhole drop {#armholedrop}

**Lower the armhole by this amount. Negative values will raise it.**

- Type: **Percentage**
- Default: **10%**
- Minimum: **0%**
- Maximum: **75%**

<Armholedrop />

### Back armhole shape {#backlinebend}

**Determines the shape/bend of the back of the armholes.**

- Type: **Percentage**
- Default: **50%**
- Minimum: **25%**
- Maximum: **100%**

<Backlinebend />

### Knit binding width {#knitbindingwidth}

**Controls the width of the knit binding as a percentage of the chosen seam allowance, or 1cm if no seam allowance is used.**

- Type: **Percentage**
- Default: **600%**
- Minimum: **300%**
- Maximum: **800%**

<Knitbindingwidth />

### Length bonus {#lengthbonus}

**The amount to lengthen the garment. A negative value will shorten it.**

- Type: **Percentage**
- Default: **10%**
- Minimum: **-20%**
- Maximum: **60%**

<Lengthbonus />

### Neckline shape {#necklinebend}

**Determines the shape/bend of the neckline at the front.**

- Type: **Percentage**
- Default: **100%**
- Minimum: **40%**
- Maximum: **100%**

<Necklinebend />

### Neckline drop {#necklinedrop}

**The amount the neck is cutout at the front.**

- Type: **Percentage**
- Default: **20%**
- Minimum: **10%**
- Maximum: **35%**

<Necklinedrop />

### Shoulderstrap placement {#shoulderstrapplacement}

**Determines whether the shoulder strap is placed closer to the neck (lower numbers) or the shoulder (higher numbers).**

- Type: **Percentage**
- Default: **40%**
- Minimum: **20%**
- Maximum: **80%**

<Shoulderstrapplacement />

### Shoulderstrap width {#shoulderstrapwidth}

**The width of the shoulder straps.**

- Type: **Percentage**
- Default: **15%**
- Minimum: **10%**
- Maximum: **40%**

<Shoulderstrapwidth />

## Advanced {#advanced}

### Legacy armhole depth {#legacyarmholedepth}

**Enable this option to use the legacy way to calculate the armhole depth (using biceps circumference) rather than the v3 way (using the waist to armpit measurement)**

- Type: **Boolean**
- Default: **false**

<Legacyarmholedepth />
