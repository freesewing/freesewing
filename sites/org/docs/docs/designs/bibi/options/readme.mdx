## Conditional Options {#conditional}

### Bust ease {#bustease}

**Ease at the bust. Only applies when bust adjustment is enabled.**

- Type: **Percentage**
- Default: **0%**
- Minimum: **0%**
- Maximum: **10%**

<Bustease />

### Waist ease {#waistease}

**Ease at the waist (only applies when the waist is fitted).**

- Type: **Percentage**
- Default: **5%**
- Minimum: **-10%**
- Maximum: **20%**

<Waistease />

### Waist adjustment {#waistadjustment}

**Controls the minimum width around the waist to prevent an excessive hourglass shape.**

- Type: **Percentage**
- Default: **95%**
- Minimum: **50%**
- Maximum: **100%**

<Waistadjustment />

### Flare {#flare}

**How much fabric to use at the bottom hem of the design. Only applies when the design goes below the waist.**

- Type: **Percentage**
- Default: **0%**
- Minimum: **0%**
- Maximum: **150%**

<Flare />

### Cuff ribbing {#usecuffribbing}

**Include ribbing on the sleeve**

- Type: **Boolean**
- Default: **false**

<Usecuffribbing />

### Sleeve fullness {#cuffease}

**Controls how wide the sleeves are.**

- Type: **Percentage**
- Default: **20%**
- Minimum: **0%**
- Maximum: **200%**

<Cuffease />

### Sleeve length {#sleevelength}

**Controls the length of your sleeves**

- Type: **Percentage**
- Default: **20%**
- Minimum: **-20%**
- Maximum: **110%**

<Sleevelength />

### Armhole curve back {#armholecurveback}

**How round the armhole curve is at the back. Higher options leave the shoulder blades free.**

- Type: **Percentage**
- Default: **30%**
- Minimum: **-10%**
- Maximum: **120%**

<Armholecurveback />

### Armhole curve front {#armholecurvefront}

**How round the armhole curve is at the front. Higher options expose more skin.**

- Type: **Percentage**
- Default: **15%**
- Minimum: **-10%**
- Maximum: **30%**

<Armholecurvefront />

### Armhole drop back {#armholedropback}

**How much the back armhole curves towards the bottom.**

- Type: **Percentage**
- Default: **20%**
- Minimum: **-50%**
- Maximum: **50%**

<Armholedropback />

### Strap width {#strapwidth}

**How wide the shoulder straps are. Only applies when sleeves are disabled.**

- Type: **Percentage**
- Default: **40%**
- Minimum: **15%**
- Maximum: **100%**

<Strapwidth />

### Ribbing height {#ribbingheight}

**Controls the height of the ribbing**

- Type: **Percentage**
- Default: **10%**
- Minimum: **5%**
- Maximum: **15%**

<Ribbingheight />

### Armhole depth {#armholedepth}

**Controls the depth of the armhole. Higher values make a deeper armhole.**

- Type: **Percentage**
- Default: **2%**
- Minimum: **-10%**
- Maximum: **50%**

<Armholedepth />

## Fit {#fit}

### Chest ease {#chestease}

**The amount of ease at your chest.**

- Type: **Percentage**
- Default: **2%**
- Minimum: **-5%**
- Maximum: **25%**

<Chestease />

### Fit the waist {#fitwaist}

**Whether or not to fit the waist.**

- Type: **Boolean**
- Default: **true**

<Fitwaist />

### Hips ease {#hipsease}

**Ease at the hips.**

- Type: **Percentage**
- Default: **5%**
- Minimum: **-5%**
- Maximum: **50%**

<Hipsease />

### Seat ease {#seatease}

**Ease at the seat.**

- Type: **Percentage**
- Default: **2%**
- Minimum: **-5%**
- Maximum: **50%**

<Seatease />

### Biceps ease {#bicepsease}

**The amount of ease at your upper arm. Note that while we try to respect this, fitting the sleeve to the armhole takes precedence over respecting the exact amount of ease.**

- Type: **Percentage**
- Default: **5%**
- Minimum: **0%**
- Maximum: **50%**

<Bicepsease />

### Allow bust dart {#dart}

**Allow creating a bust dart if this is necessary**

- Type: **Boolean**
- Default: **false**

<Dart />

### Bust adjustment {#draftforhighbust}

**If the pattern should be drafted with a basic full bust adjustment (FBA). This will result in a more fitted garment for people with breasts.**

- Type: **Boolean**
- Default: **true**

<Draftforhighbust />

### Ribbing stretch {#ribbingstretch}

**Use this to adapt the pattern to how stretchy the ribbing that you are using is.**

- Type: **Percentage**
- Default: **15%**
- Minimum: **0%**
- Maximum: **30%**

<Ribbingstretch />

### Shoulder ease {#shoulderease}

**The amount of ease at your shoulder. This increases the shoulder to shoulder distance to accommodate additional layers or thickness.**

- Type: **Percentage**
- Default: **0%**
- Minimum: **-2%**
- Maximum: **6%**

<Shoulderease />

## Style {#style}

### Back neckline curvature {#backneckbend}

**Controls the curvature of the back neckline.**

- Type: **Percentage**
- Default: **50%**
- Minimum: **0%**
- Maximum: **70%**

<Backneckbend />

### Back neckline depth {#backneckcutout}

**Controls how deep the neck opening plunges down on the back.**

- Type: **Percentage**
- Default: **6%**
- Minimum: **2%**
- Maximum: **110%**

<Backneckcutout />

### Knit binding height {#bindingheight}

**Controls the height of the knit binding for neck and armhole.**

- Type: **Percentage**
- Default: **2.5%**
- Minimum: **1.25%**
- Maximum: **5%**

<Bindingheight />

### Front neckline curvature {#necklinebend}

**Controls the curvature of the front neckline.**

- Type: **Percentage**
- Default: **50%**
- Minimum: **0%**
- Maximum: **70%**

<Necklinebend />

### Front neckline depth {#necklinedepth}

**Controls how deep the neck opening plunges down at the front.**

- Type: **Percentage**
- Default: **25%**
- Minimum: **20%**
- Maximum: **110%**

<Necklinedepth />

### Neckline width {#necklinewidth}

**Controls the width of the neck opening.**

- Type: **Percentage**
- Default: **15%**
- Minimum: **-5%**
- Maximum: **90%**

<Necklinewidth />

### Waistband ribbing {#usewaistribbing}

**Include ribbing on the waistband**

- Type: **Boolean**
- Default: **false**

<Usewaistribbing />

### Length {#length}

#### Length {#length}

**Which measurement line to use for the bottom hem. You can do fine adjustments using the bonus length option.**

- Type: **List**
- Default: **seat**
- options:
  - underbust
  - waist
  - hips
  - seat
  - knee
  - floor

<Length />

#### Length bonus {#lengthbonus}

**The amount to lengthen the garment. A negative value will shorten it.**

- Type: **Percentage**
- Default: **0%**
- Minimum: **-30%**
- Maximum: **30%**

<Lengthbonus />

### Sleeves {#sleeves}

#### Sleeves {#sleeves}

**Draft the pattern with sleeves**

- Type: **Boolean**
- Default: **true**

<Sleeves />

## Advanced {#advanced}

### Across back factor {#acrossbackfactor}

**Controls your across back width as a factor of your shoulder to shoulder measurement.**

- Type: **Percentage**
- Default: **98%**
- Minimum: **93%**
- Maximum: **100%**

<Acrossbackfactor />

### Curvature adjustment {#curvatureadjustment}

**Controls how straight the side seam is above the hem.**

- Type: **Percentage**
- Default: **50%**
- Minimum: **0.1%**
- Maximum: **100%**

<Curvatureadjustment />

### Length adjustment {#lengthadjustment}

**Controls how much the front part is made longer to compensate for the curve around the bust. Has little effect on smaller busts.**

- Type: **Percentage**
- Default: **25%**
- Minimum: **0%**
- Maximum: **100%**

<Lengthadjustment />

### Seat back adjustment {#seatbackadjustment}

**Controls the influence of the seat back measurement.**

- Type: **Percentage**
- Default: **20%**
- Minimum: **0%**
- Maximum: **100%**

<Seatbackadjustment />
