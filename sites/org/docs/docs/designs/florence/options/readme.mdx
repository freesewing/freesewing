## Fit {#fit}
### Curve {#curve}

**Controls the curvature of the upper edge of the face mask**
- Type: **Percentage**
- Default: **12.5%**
- Minimum: **10%**
- Maximum: **15%**

### Height {#height}

**Controls the height of the face mask**
- Type: **Percentage**
- Default: **26%**
- Minimum: **23%**
- Maximum: **29%**

<Height />

### Length {#length}

**Controls the length of the face mask**
- Type: **Percentage**
- Default: **40%**
- Minimum: **35%**
- Maximum: **45%**
