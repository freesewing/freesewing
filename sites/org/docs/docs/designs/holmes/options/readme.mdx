## Fit {#fit}

### Head ease {#headease}

**The amount of ease around your head.**

- Type: **Percentage**
- Default: **3%**
- Minimum: **0%**
- Maximum: **9%**

<Headease />

## Style {#style}

### Buttonhole guide {#buttonhole}

**Adds a buttonhole to the ear flap to help you draft the buttonhole ear flap variant**

- Type: **Boolean**
- Default: **false**

<Buttonhole />

### Ear flap length {#earlength}

**Controls the length of the ear flaps independently from the crown pieces**

- Type: **Percentage**
- Default: **100%**
- Minimum: **80%**
- Maximum: **150%**

<Earlength />

### Ear flap width {#earwidth}

**Controls the width of the ear flaps**

- Type: **Percentage**
- Default: **100%**
- Minimum: **80%**
- Maximum: **150%**

<Earwidth />

### Number of gores {#gores}

**The number of gores used to construct the crown**

- Type: **Percentage**
- Default: **undefined**
- Minimum: **4**
- Maximum: **20**

<Gores />

### Length ratio {#lengthratio}

**Controls the length of the crown and ear flaps**

- Type: **Percentage**
- Default: **55%**
- Minimum: **40%**
- Maximum: **60%**

<Lengthratio />

### Visor angle {#visorangle}

**The arc angle used to draft the inner curve of the visor**

- Type: **Degrees**
- Default: **45°**
- Minimum: **10°**
- Maximum: **90°**

<Visorangle />

### Visor width {#visorwidth}

**Controls the width of the visor**

- Type: **Percentage**
- Default: **5%**
- Minimum: **1%**
- Maximum: **17%**

<Visorwidth />

## Advanced {#advanced}

### Visor length {#visorlength}

**Controls the length of the visor**

- Type: **Percentage**
- Default: **100%**
- Minimum: **80%**
- Maximum: **150%**

<Visorlength />
