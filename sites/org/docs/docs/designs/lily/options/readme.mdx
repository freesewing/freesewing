## Fit {#fit}
### Fabric stretch {#fabricstretch}

**How much the fabric can stretch horizontally; this is used to calculate ease**
- Type: **Percentage**
- Default: **40%**
- Minimum: **0%**
- Maximum: **50%**

<Fabricstretch />

### Knee ease {#kneeease}

**Controls the amount of ease at the knee**
- Type: **Percentage**
- Default: **-4%**
- Minimum: **-20%**
- Maximum: **0%**

<Kneeease />

### Seat ease {#seatease}

**Controls the amount of ease at your seat**
- Type: **Percentage**
- Default: **-4%**
- Minimum: **-20%**
- Maximum: **0%**

<Seatease />

### Waist ease {#waistease}

**Controls the amount of ease at your waist**
- Type: **Percentage**
- Default: **-4%**
- Minimum: **-20%**
- Maximum: **0%**

<Waistease />

## Style {#style}
### Length reduction {#lengthreduction}

**Controls how much the leggings are shortened with respect to ankle length**
- Type: **Percentage**
- Default: **0%**
- Minimum: **0%**
- Maximum: **100%**

<Lengthreduction />

### Use waistband angle for {#usewaistanglefor}

**Apply the waistband angle to the front, back or both**
- Type: **List**
- Default: **both**
- options:
  - both
  - backOnly
  - frontOnly

<Usewaistanglefor />

### Waistband angle {#waistangle}

**Change the angle of the waistband when viewed from the side**
- Type: **Degrees**
- Default: **0°**
- Minimum: **-20°**
- Maximum: **20°**

<Waistangle />

### Waist height {#waistheight}

**Controls the height of the waist, 100% = waist height, 0% = hip height**
- Type: **Percentage**
- Default: **50%**
- Minimum: **0%**
- Maximum: **100%**

<Waistheight />

### Waistband width {#waistbandwidth}

**The width of the waistband**
- Type: **Percentage**
- Default: **3%**
- Minimum: **1%**
- Maximum: **6%**

<Waistbandwidth />

## Advanced {#advanced}
### Cross seam angle {#crossseamcurveangle}

**Controls the angle of the cross seam**
- Type: **Degrees**
- Default: **12°**
- Minimum: **0°**
- Maximum: **20°**

<Crossseamcurveangle />

### Cross seam bend {#crossseamcurvebend}

**Controls the curvature of the cross seam**
- Type: **Percentage**
- Default: **65%**
- Minimum: **45%**
- Maximum: **85%**

<Crossseamcurvebend />

### Start of the cross seam curve {#crossseamcurvestart}

**Controls how far into the cross seam we start to curve**
- Type: **Percentage**
- Default: **85%**
- Minimum: **60%**
- Maximum: **100%**

<Crossseamcurvestart />

### Crotch drop {#crotchdrop}

**Lowers the crotch for a more relaxed fit**
- Type: **Percentage**
- Default: **0%**
- Minimum: **0%**
- Maximum: **15%**

<Crotchdrop />

### Crotch seam angle {#crotchseamcurveangle}

**Controls the angle of the crotch seam**
- Type: **Degrees**
- Default: **25°**
- Minimum: **0°**
- Maximum: **35°**

<Crotchseamcurveangle />

### Crotch seam bend {#crotchseamcurvebend}

**Controls the curvature of the crotch seam**
- Type: **Percentage**
- Default: **80%**
- Minimum: **45%**
- Maximum: **100%**

<Crotchseamcurvebend />

### Start of the crotch seam curve {#crotchseamcurvestart}

**Controls how far into the crotch seam we start to curve**
- Type: **Percentage**
- Default: **80%**
- Minimum: **60%**
- Maximum: **95%**

<Crotchseamcurvestart />

### Show measurement guides {#fitguides}

**Show measurement guides for horizontal measures**
- Type: **Boolean**
- Default: **false**

<Fitguides />

### Grainline position {#grainlineposition}

**Controls the horizontal position of the leg relative to the seat**
- Type: **Percentage**
- Default: **45%**
- Minimum: **30%**
- Maximum: **60%**

<Grainlineposition />
