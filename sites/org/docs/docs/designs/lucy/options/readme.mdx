## Style {#style}
### Taper {#edge}

**Controls how much the pocket opening tapers inwards**
- Type: **Percentage**
- Default: **25%**
- Minimum: **20%**
- Maximum: **50%**

<Edge />

### Length {#length}

**Length (depth) of the pocket**
- Type: **Percentage**
- Default: **50%**
- Minimum: **30%**
- Maximum: **100%**

<Length />

### Width {#width}

**Width of the pocket**
- Type: **Percentage**
- Default: **50%**
- Minimum: **30%**
- Maximum: **100%**

<Width />
