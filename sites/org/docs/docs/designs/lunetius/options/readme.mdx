## Style {#style}
### Length {#length}

**Choose from the different length styles**
- Type: **List**
- Default: **toBelowKnee**
- options:
  - toKnee
  - toBelowKnee
  - toHips
  - toUpperLeg
  - toFloor

### Length ratio {#lengthratio}

**Controls the length of the garment**
- Type: **Percentage**
- Default: **105%**
- Minimum: **60%**
- Maximum: **130%**

<Lengthratio />

### Width ratio {#widthratio}

**Controls the width of the garment**
- Type: **Percentage**
- Default: **100%**
- Minimum: **50%**
- Maximum: **130%**
