## Construction {#construction}

### Hem width {#hemwidth}

**Width of the hem**

- Type: **Percentage**
- Default: **2%**
- Minimum: **1%**
- Maximum: **10%**

<Hemwidth />

### Panels {#panels}

**Number of panels**

- Type: **Percentage**
- Default: **undefined**
- Minimum: **1**
- Maximum: **8**

<Panels />

### Seamless full circle {#seamlessfullcircle}

**Enables a seamless full circle skirt.**

- Type: **Boolean**
- Default: **false**

<Seamlessfullcircle />

## Fit {#fit}

### Waistband position {#waistbandposition}

**Controls the position of the waistband.**

- Type: **Percentage**
- Default: **50%**
- Minimum: **0%**
- Maximum: **100%**

<Waistbandposition />

### Waistband shape {#waistbandshape}

**Whether you want a straight or shaped waistband.**

- Type: **List**
- Default: **straight**
- options:
  - straight
  - curved

<Waistbandshape />

## Style {#style}

### Circle ratio {#circleratio}

**The percentage of a circle you want the skirt to be.**

- Type: **Percentage**
- Default: **50%**
- Minimum: **20%**
- Maximum: **100%**

<Circleratio />

### Gathering {#gathering}

**The percent by which the top of the skirt is longer than the bottom of the waistband.**

- Type: **Percentage**
- Default: **0%**
- Minimum: **0%**
- Maximum: **200%**

<Gathering />

### Length bonus {#lengthbonus}

**Determines how long the skirt will be.**

- Type: **Percentage**
- Default: **50%**
- Minimum: **10%**
- Maximum: **100%**

<Lengthbonus />

### Waistband overlap {#waistbandoverlap}

**The amount by which the waistband overlaps.**

- Type: **Percentage**
- Default: **3%**
- Minimum: **0%**
- Maximum: **15%**

<Waistbandoverlap />

### Waistband width {#waistbandwidth}

**Controls the width of the waistband.**

- Type: **Percentage**
- Default: **4%**
- Minimum: **1%**
- Maximum: **8%**

<Waistbandwidth />
