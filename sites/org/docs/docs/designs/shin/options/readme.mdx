## Fit {#fit}
### Back rise {#backrise}

**Controls how much the waist is higher in the back than  the front.**
- Type: **Percentage**
- Default: **5%**
- Minimum: **0%**
- Maximum: **10%**

<Backrise />

### Bulge {#bulge}

**Controls the amount of extra room to create at the front of the trunks.**
- Type: **Percentage**
- Default: **2.5%**
- Minimum: **0%**
- Maximum: **5%**

<Bulge />

### Leg reduction {#legreduction}

**Reduces the leg opening to prevent gaping**
- Type: **Percentage**
- Default: **5%**
- Minimum: **0%**
- Maximum: **10%**

<Legreduction />

### Stretch {#stretch}

**The amount of stretch to design for.**
- Type: **Percentage**
- Default: **20%**
- Minimum: **10%**
- Maximum: **30%**

<Stretch />

## Style {#style}
### Elastic width {#elasticwidth}

**Width of the elastic at the waist**
- Type: **Percentage**
- Default: **10%**
- Minimum: **4%**
- Maximum: **20%**

<Elasticwidth />

### Rise {#rise}

**Controls the height of the waist.**
- Type: **Percentage**
- Default: **0%**
- Minimum: **0%**
- Maximum: **25%**

<Rise />
