## Fit {#fit}

### Knee ease {#kneeease}

**Controls the amount of ease at the knee**

- Type: **Percentage**
- Default: **6%**
- Minimum: **1%**
- Maximum: **25%**

<Kneeease />

### Seat ease {#seatease}

**Controls the amount of ease at your seat**

- Type: **Percentage**
- Default: **2%**
- Minimum: **0%**
- Maximum: **10%**

<Seatease />

### Waist ease {#waistease}

**Controls the amount of ease at your waist**

- Type: **Percentage**
- Default: **2%**
- Minimum: **0%**
- Maximum: **10%**

<Waistease />

## Style {#style}

### Crotch drop {#crotchdrop}

**Lowers the crotch for a more relaxed fit**

- Type: **Percentage**
- Default: **2%**
- Minimum: **0%**
- Maximum: **15%**

<Crotchdrop />

### Fit the knee {#fitknee}

**Fits the legs from based on the knee circumference, rather than seat circumference**

- Type: **Boolean**
- Default: **false**

<Fitknee />

### Length bonus {#lengthbonus}

**Controls the length of the trousers**

- Type: **Percentage**
- Default: **2%**
- Minimum: **-20%**
- Maximum: **10%**

<Lengthbonus />

### Use waistband angle for {#usewaistanglefor}

**Apply the waistband angle to the front, back or both**

- Type: **List**
- Default: **both**
- options:
  - both
  - backOnly
  - frontOnly

<Usewaistanglefor />

### Waistband angle {#waistangle}

**Change the angle of the waistband when viewed from the side**

- Type: **Degrees**
- Default: **0°**
- Minimum: **-20°**
- Maximum: **20°**

<Waistangle />

### Waist height {#waistheight}

**Controls the height of the waist, 100% = waist height, 0% = hip height**

- Type: **Percentage**
- Default: **100%**
- Minimum: **0%**
- Maximum: **100%**

<Waistheight />

## Advanced {#advanced}

### Cross seam angle {#crossseamcurveangle}

**Controls the angle of the cross seam**

- Type: **Degrees**
- Default: **12°**
- Minimum: **0°**
- Maximum: **20°**

<Crossseamcurveangle />

### Cross seam bend {#crossseamcurvebend}

**Controls the curvature of the cross seam**

- Type: **Percentage**
- Default: **65%**
- Minimum: **45%**
- Maximum: **85%**

<Crossseamcurvebend />

### Start of the cross seam curve {#crossseamcurvestart}

**Controls how far into the cross seam we start to curve**

- Type: **Percentage**
- Default: **85%**
- Minimum: **60%**
- Maximum: **100%**

<Crossseamcurvestart />

### Crotch seam angle {#crotchseamcurveangle}

**Controls the angle of the crotch seam**

- Type: **Degrees**
- Default: **25°**
- Minimum: **0°**
- Maximum: **35°**

<Crotchseamcurveangle />

### Crotch seam bend {#crotchseamcurvebend}

**Controls the curvature of the crotch seam**

- Type: **Percentage**
- Default: **80%**
- Minimum: **45%**
- Maximum: **100%**

<Crotchseamcurvebend />

### Start of the crotch seam curve {#crotchseamcurvestart}

**Controls how far into the crotch seam we start to curve**

- Type: **Percentage**
- Default: **80%**
- Minimum: **60%**
- Maximum: **95%**

<Crotchseamcurvestart />

### Grainline position {#grainlineposition}

**Controls the horizontal position of the leg relative to the seat**

- Type: **Percentage**
- Default: **45%**
- Minimum: **30%**
- Maximum: **60%**

<Grainlineposition />

### Leg balance {#legbalance}

**Controls the ratio between front and back panel of the leg**

- Type: **Percentage**
- Default: **57.5%**
- Minimum: **52.5%**
- Maximum: **62.5%**

<Legbalance />

### Waist balance {#waistbalance}

**Controls the horizontal position of the waist relative to the seat**

- Type: **Percentage**
- Default: **60%**
- Minimum: **30%**
- Maximum: **90%**

<Waistbalance />

### Waistband width {#waistbandwidth}

**The width of the waistband**

- Type: **Percentage**
- Default: **3%**
- Minimum: **1%**
- Maximum: **6%**

<Waistbandwidth />
