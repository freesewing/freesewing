## Fit {#fit}

### Bulge {#bulge}

**Increase the bugle angle to create more room in the front pouch. Doesn't kick in below 2 degrees.**

- Type: **Degrees**
- Default: **0°**
- Minimum: **0°**
- Maximum: **30°**

<Bulge />

### Gusset length {#gussetlength}

**Controls the length of the gusset**

- Type: **Percentage**
- Default: **12.7%**
- Minimum: **10%**
- Maximum: **16%**

<Gussetlength />

### Gusset position {#gussetposition}

**This option allows you to shift the gusset more to the front or back. It is ignored when bulge is active**

- Type: **Percentage**
- Default: **70%**
- Minimum: **5%**
- Maximum: **95%**

<Gussetposition />

### Gusset width {#gussetwidth}

**Controls the width of the gusset**

- Type: **Percentage**
- Default: **15%**
- Minimum: **5%**
- Maximum: **24%**

<Gussetwidth />

### Horizontal fabric stretch {#xstretch}

**Adjust this for more or less stretchy fabric in the horizontal direction**

- Type: **Percentage**
- Default: **15%**
- Minimum: **0%**
- Maximum: **30%**

<Xstretch />

### Vertical fabric stretch {#ystretch}

**Adjust this for more or less stretchy fabric in the vertical direction**

- Type: **Percentage**
- Default: **5%**
- Minimum: **0%**
- Maximum: **15%**

<Ystretch />

## Style {#style}

### Back waist dip {#backdip}

**Controls how much the back waist curves (revealing more or less skin)**

- Type: **Percentage**
- Default: **-5%**
- Minimum: **-15%**
- Maximum: **10%**

<Backdip />

### Back exposure {#backexposure}

**Controls the amount of exposed skin on the back**

- Type: **Percentage**
- Default: **30%**
- Minimum: **0%**
- Maximum: **115%**

<Backexposure />

### Front waist dip {#frontdip}

**Controls how much the front waist curves (revealing more or less skin)**

- Type: **Percentage**
- Default: **10%**
- Minimum: **0%**
- Maximum: **25%**

<Frontdip />

### Front exposure {#frontexposure}

**Controls the amount of exposed skin on the front**

- Type: **Percentage**
- Default: **70%**
- Minimum: **5%**
- Maximum: **100%**

<Frontexposure />

### Leg rise {#legrise}

**Controls how high the leg is cut out**

- Type: **Percentage**
- Default: **54%**
- Minimum: **5%**
- Maximum: **95%**

<Legrise />

### Rise {#rise}

**Controls the height of the waist**

- Type: **Percentage**
- Default: **50%**
- Minimum: **30%**
- Maximum: **100%**

<Rise />
