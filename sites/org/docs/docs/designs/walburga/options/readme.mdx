## Fit {#fit}

### Head ratio {#headratio}

**Controls the size of the head opening**

- Type: **Percentage**
- Default: **100%**
- Minimum: **80%**
- Maximum: **120%**

<Headratio />

## Style {#style}

### Length {#length}

**Controls the length of the garment**

- Type: **List**
- Default: **toKnee**
- options:
  - toKnee
  - toMidLeg
  - toFloor

<Length />

### Length bonus {#lengthbonus}

**Allows variation of the length of the garment**

- Type: **Percentage**
- Default: **85%**
- Minimum: **60%**
- Maximum: **130%**

<Lengthbonus />

### Neckline {#neckline}

**Controls whether or not to draft a neck opening**

- Type: **Boolean**
- Default: **true**

<Neckline />

### Neck opening shape {#neckoratio}

**Controls the shape of the neck opening**

- Type: **Percentage**
- Default: **100%**
- Minimum: **10%**
- Maximum: **190%**

<Neckoratio />

### Width bonus {#widthbonus}

**Allows variation of the width of the garment**

- Type: **Percentage**
- Default: **95%**
- Minimum: **50%**
- Maximum: **130%**

<Widthbonus />
