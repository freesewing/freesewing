## Style {#style}
### Aggressive {#aggressive}

**Give Hi pointy teeth, or not**
- Type: **Boolean**
- Default: **false**

<Aggressive />

### Hungry {#hungry}

**Changes the belly shape to convey if Hi has just eaten**
- Type: **Percentage**
- Default: **50%**
- Minimum: **0%**
- Maximum: **100%**

<Hungry />

### Nose pointiness {#nosepointiness}

**Controls how pointy Hi's nose is**
- Type: **Percentage**
- Default: **0%**
- Minimum: **-5%**
- Maximum: **10%**

<Nosepointiness />

### Size {#size}

**Sharks come in all sizes, and so does Hi**
- Type: **Percentage**
- Default: **100%**
- Minimum: **5%**
- Maximum: **500%**

<Size />
